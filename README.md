## 주제 선정 및 배경   
- 카드사의 주요 수익원은 회원 연회비 및 가맹점 수수료
- 아래의 이유로 카드사의 수익구조가 무너짐과 동시에 성장 한계에 직면함   
  1)경제 성장률 둔화   
  2)가맹점 수수료 인하   
  3)조달비율 상승   
  4)금리 상한 인하   
  5)정부의 보수적 규제 장기화   
  6)제 2금융 대출 규제 강화   
  7)핀테크 활성화
